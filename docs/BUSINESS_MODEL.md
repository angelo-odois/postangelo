# Revuu - Business Model & Monetizacao

> Documento completo com estrategias de monetizacao e modelo de negocios.

---

## Visao Geral

O Revuu sera uma plataforma hibrida que combina:
1. **Portfolio Builder** (B2C) - Criacao de portfolios profissionais
2. **Talent Marketplace** (B2B2C) - Conexao entre empresas e talentos
3. **HR Tech** (B2B) - Ferramentas de recrutamento estilo Gupy

## 16. Diferenciais vs Concorrencia

### vs Gupy

| Revuu | Gupy |
|-------|------|
| Portfolio visual | Curriculo texto |
| Comunidade ativa | So recrutamento |
| Nicho (tech/design) | Generalista |
| Self-service | Enterprise-first |
| Success fee | Assinatura fixa |
| Menor ticket | Alto ticket |

### vs LinkedIn

| Revuu | LinkedIn |
|-------|----------|
| Portfolio rico | Perfil simples |
| Foco em criativos | Generalista |
| Comunidade local | Global |
| Preco acessivel | Premium caro |
| UX moderna | UX datada |

### vs Behance/Dribbble

| Revuu | Behance/Dribbble |
|-------|------------------|
| Multi-nicho | So design |
| Recrutamento integrado | Sem job board robusto |
| Monetizacao para usuarios | Limitada |
| Mercado BR | Foco global |
